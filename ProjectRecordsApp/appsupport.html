Project Records version:1.0

This is an app that helps you keep track of your work items, and can record project information, project schedule information, and project earning information.


1. Project information:

You can save the project's name, type, project content, contacts, contact phone number and other information, you can click on the project list and then add or view the project schedule, project earning information;


2. Project Schedule:

You can select specific projects and fill in project schedule information: completed project content, unfinished content, current project schedule; you can also view a specific project schedule list;


3. Project Earning:

After selecting a specific projects, you can view the earning information of the item: payment amount, payment time, payment order;


4. More:

You can view a report of project payments based on the type of project. The report is displayed in a pie chart and is simple and intuitive.
